# NFC-Door (English)

> Requirements:

- Arduino UNO board
- Arduino 9V Power Supply
- RC522 RFID Module
- 2 LED's (Green, Red)
- SG90 Servomotor
- (Optional) Jumpers

> What type of NFC works?

- RFID Cards
- RFID Tags
- Phone NFC

# NFC-Door (Español)

> Requisitos:

- Placa Arduino UNO
- Alimentación para Arduino de 9V
- Modulo RFID RC522
- 2 LED's (Verde, Rojo)
- Servomotor SG90
- (Opcional) Cables Dupont

> ¿Que tipo de NFC's funcionan?

- Tarjetas RFID
- Llaveros RFID
- NFC de un dispositivo móvil
